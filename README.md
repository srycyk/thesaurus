
# A Thesaurus

This is the first stab at a one-page web application to show
links between words. It is briefly described in the partial,
**app/views/definitions/_splash.html.haml**.

The English database, whose size is over 200M in Sqlite, is not
included.
It is created from a few (open) sources and takes a good while,
and several steps, to build.
Once built it remains intact - which makes the application very
low maintenance!
